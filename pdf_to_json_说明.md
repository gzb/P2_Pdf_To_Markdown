# PDF 到 JSON 及图像处理工具说明 (pdf_to_json.py)

本工具是一个基于 Python 的 PDF 页面级处理脚本，主要利用 `PyMuPDF (fitz)` 和 `Pillow (PIL)` 库，实现将 PDF 文件按页拆解、提取文本区块信息，并进行图像渲染与可视化标注。

## 核心功能

1.  **PDF 页面数据提取 (`extract_pdf_to_json`)**:
    *   **逐页解析**: 打开指定的 PDF 文件，并逐页遍历。
    *   **区块提取**: 使用 PyMuPDF 的 `blocks` 模式提取每一页的文本块信息。
    *   **JSON 序列化**: 过滤掉非文本和空文本块，将每页的文本内容 (`text`) 和其对应的精准坐标 (`bbox: [x0, y0, x1, y1]`) 提取出来，保存为独立的 JSON 文件（例如 `1.json`, `2.json`）。
    *   **页面基础信息**: JSON 文件中同时记录了该页的原始宽度 (`width`) 和高度 (`height`)。

2.  **PDF 页面图像渲染**:
    *   在提取 JSON 数据的同时，将 PDF 每一页渲染为 PNG 图像。
    *   为了保证图像清晰度，采用了 2 倍缩放矩阵 (`fitz.Matrix(2, 2)`) 进行高清渲染。
    *   生成的图像按页码命名并保存在独立的目录中。

3.  **文本区块可视化标注 (`draw_boxes_on_images`)**:
    *   **数据映射**: 遍历已生成的页面图像文件，并读取对应页码的 JSON 数据文件。
    *   **坐标转换**: 自动计算 PDF 原始尺寸与高清渲染图像尺寸之间的缩放比例 (`scale_factor`)，将 JSON 中的 PDF 坐标精准映射到图片像素坐标。
    *   **红框绘制**: 使用 `Pillow` 库在图像上为每一个文本区块绘制红色矩形框 (`outline="red", width=2`)，直观展示文本提取的空间位置。
    *   **结果保存**: 将带有红框标注的图片保存到新的目录中，方便人工核对和调试。

## 使用方法

### 依赖安装
运行此脚本需要安装以下 Python 库：
```bash
pip install pymupdf pillow
```

### 运行脚本
直接运行该 Python 文件即可执行处理流程：
```bash
python pdf_to_json.py
```

### 目录结构说明
在默认配置下，脚本会处理指定的输入文件并生成三个输出目录：
*   **输入文件**: `test_data/人民法院思想政治建设读本3.7_2026_03_09_13_53_13.pdf`
*   **输出 JSON 目录 (`out_folder`)**: `test_data/pdf-2-json-pyhton` (存放每页提取的坐标和文本数据)
*   **输出高清图像目录 (`img_folder`)**: `test_data/pdf-1-images` (存放 PDF 渲染的原始图片)
*   **输出标注图像目录 (`marked_img_folder`)**: `test_data/pdf-1-images-python` (存放带有红色区块标注的图片)

*注：在实际使用时，可以通过注释或取消注释 `__main__` 块中的函数调用，来单独执行“提取数据”或“绘制红框”步骤。*
