# PDF to Markdown 转换与交互系统功能说明

本系统是一个基于 B/S 架构的智能图书稿处理平台，旨在实现 PDF 到 Markdown 的高质量转换，并提供深度的图文交互体验。

## 核心功能

### 1. 智能格式转换
- **PDF 内容分析**: 使用 PyMuPDF (fitz) 深度解析 PDF 结构。
- **Markdown 生成**: 自动识别标题层级（基于字体大小启发式算法）并生成标准的 Markdown 文本。
- **坐标映射**: 系统在转换过程中精确记录每个文本块在 PDF 中的**页码**和**坐标 (x0, y0, x1, y1)**，并保存为 JSON 映射文件。

### 2. 双栏同步交互预览
- **高清 PDF 展示**: 基于 PDF.js 实现，支持 High DPI 渲染，确保文字清晰。
- **连续滚动模式**: PDF 展示区域支持全文档连续滚动，无需频繁手动翻页。
- **点击跳转定位**: 
    - 点击右侧 Markdown 的任意行，左侧 PDF 会自动平滑滚动到对应位置。
    - 命中区域会以**半透明红色块**进行标注，并在左上角标记**小红旗 (🚩)** 图标以供快速定位。

### 3. 高级搜索与定位
- **全文关键词检索**: 后端实时在 PDF 原始数据中搜索关键词。
- **搜索摘要高亮**: 搜索结果在右侧以摘要形式展示，摘要中的关键词进行**红字标粗**。
- **精准定位**: 点击搜索结果，左侧 PDF 会精确高亮关键词所在的具体词语位置，而非整行高亮。

### 4. 划区文本提取
- **选取模式**: 提供 "Enable Selection" 模式。
- **鼠标划区**: 用户可以在 PDF 展示区通过鼠标拖拽画出蓝色虚线框。
- **实时提取**: 系统自动识别框选区域内的 PDF 文本内容。
- **浮动展示**: 提取的文本显示在一个可关闭的浮动图层中，方便用户快速摘录或对比。

### 5. 视图控制
- **缩放功能**: 支持 PDF 区域的放大与缩小，且高亮标注会随缩放自动重新计算位置，保持精准。
- **快速跳转**: 提供页码输入框，支持一键跳转到指定页面。

## 技术栈
- **后端**: Python, FastAPI, PyMuPDF (fitz), Uvicorn.
- **前端**: HTML5, CSS3 (Flexbox/Fixed Layout), JavaScript (ES6+), PDF.js.

## 项目结构
- `backend/main.py`: 后端 API 逻辑。
- `backend/processor.py`: PDF 解析与 Markdown 转换逻辑。
- `backend/static/`: 前端资源（HTML, App.js, CSS）。
- `backend/uploads/`: 存储上传的原始文件及处理生成的中间数据。
